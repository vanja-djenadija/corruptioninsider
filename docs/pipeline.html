📌 Legend & Key Information

  • 🔴 CRITICAL Risk: Immediate attention required (e.g., single bidder, sanctioned supplier)
  • 🟠 HIGH Risk: Significant concerns (e.g., 2 bidders, negotiated procedure)
  • 🟡 MEDIUM Risk: Moderate concerns (e.g., 3 bidders, pre-election timing)
  • ✅ LOW/OK: No significant red flags detected
  • ⚡ LOAD DATA INFILE: MySQL bulk import - 50-100x faster than row-by-row
  • 📊 SQL Views: Pre-aggregated queries for fast dashboard loading
  • 🔄 Cron Jobs: Automated scheduling (daily/weekly/monthly)
